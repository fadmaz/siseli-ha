# Документація PowMr Bridge

## Налаштування інвертора
Щоб інвертор почав надсилати дані в цей додаток, потрібно, щоб він "думав", що надсилає їх у хмару Siseli.

### Варіант А: Налаштування DNS (Рекомендовано)
Налаштуйте ваш роутер так, щоб запити до домену хмари PowMr резолвились на локальну IP-адресу вашого Home Assistant.

### Варіант Б: Static Route
Створіть статичний маршрут на роутері, який перенаправляє трафік для IP `8.212.18.157` на IP вашого Home Assistant.

## MQTT
Додаток автоматично підключається до внутрішнього брокера `core-mosquitto`. Переконайтеся, що у вас встановлено офіційний аддон Mosquitto Broker.
